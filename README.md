# LangMark Learning Portal

A modern learning portal scaffold for Traditional Chinese language learners.

## Features

- React + TypeScript + Vite
- Tailwind CSS for polished UI
- Interactive dashboard layout
- Beginner-friendly lesson cards
- Progress and habit tracking components

## Getting Started

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production:

```bash
npm run build
```
