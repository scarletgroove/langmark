# LangMark Learning Portal

## Project Purpose
A modern learning portal for Traditional Chinese designed to help users progress from beginner to fluent with interactive lessons, daily practice, and polished UX.

## Current Scope
- React + TypeScript + Vite frontend
- Tailwind CSS for styling
- Routing with React Router
- Dashboard, curriculum, onboarding, and lesson overview pages
- Sample data for lessons and curriculum levels

## Next Priorities
- Add lesson detail screens and interactive practice components
- Add onboarding forms with personalized learning plans
- Add audio pronunciation and stroke order practice
- Integrate progress persistence and spaced repetition review
